# 16-Week Daily Micro-Lesson Plan

## Weeks 1-2: WordPress Core
- Day 1: Read WordPress Core architecture
- Day 2: Hooks & Filters
- Day 3: WP_Query & loops
- Day 4: Custom Post Types & Taxonomies
- Day 5: Build a small plugin
- Day 6-7: Review & notes

## Weeks 3-4: PHP & JS
- Day 1: PHP OOP basics
- Day 2: PHP Namespaces & Exceptions
- Day 3: JS ES6 syntax
- Day 4: AJAX in WordPress
- Day 5: jQuery practice in themes
- Day 6-7: Mini project combining PHP + JS

## Weeks 5-6: WooCommerce Architecture
- Day 1: WooCommerce templates overview
- Day 2: WooCommerce hooks & filters
- Day 3: Checkout customization basics
- Day 4: Product page customization
- Day 5: Custom product types
- Day 6-7: Build small WooCommerce plugin

## Weeks 7-8: Performance Optimization
- Day 1: Caching concepts
- Day 2: WP Rocket setup
- Day 3: CDN integration
- Day 4: Media offload & lazy loading
- Day 5: Asset optimization (minify, defer)
- Day 6-7: Practice optimization on test store

## Weeks 9-10: Profiling & Monitoring
- Day 1: Query Monitor setup
- Day 2: New Relic overview
- Day 3: Lighthouse & GTmetrix testing
- Day 4: Identify slow queries & scripts
- Day 5: Optimize database queries
- Day 6-7: Test changes & record results

## Weeks 11-12: Debugging & Conflicts
- Day 1: Theme conflict detection
- Day 2: Plugin conflict detection
- Day 3: Debugging checkout issues
- Day 4: Debugging performance issues
- Day 5: Document solutions
- Day 6-7: Apply fixes to practice store

## Weeks 13-14: Advanced Optimization
- Day 1: Server-level caching (Redis, Memcached)
- Day 2: Object caching in WooCommerce
- Day 3: Advanced asset optimization
- Day 4: Preloading & critical CSS
- Day 5: Lazy loading & responsive images
- Day 6-7: Benchmark store performance

## Weeks 15-16: Real-World Projects
- Rebuild a WooCommerce store fully optimized
- Apply all caching, CDN, asset, and performance best practices
- Track Core Web Vitals, Lighthouse, and GTmetrix scores
- Build a custom plugin for checkout or products
- Document all learnings
